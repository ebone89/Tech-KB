## üß™ Usage Examples

```c
short s = 32000;
unsigned int u = 4294967295U;
long l = 1000000000L;
long long big = 9223372036854775807LL;
```

- Add `U` for unsigned, `L` for long, `LL` for long long when using constants.

---

## ‚úÖ When to Use

- Use `short` to save memory when values are small.
- Use `unsigned` if negative values are not needed (e.g., counting items).
- Use `long` or `long long` for very large numbers.

---

## ‚ö†Ô∏è Caution

- Overflowing a signed integer causes **undefined behavior**.
- Mixing signed and unsigned types can lead to **unexpected results**.
