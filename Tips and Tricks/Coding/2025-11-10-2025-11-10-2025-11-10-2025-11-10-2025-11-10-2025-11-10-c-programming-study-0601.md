### Cups to Liters Conversion
```c

    const double CUPS_TO_LITERS = 0.236588;
    return numCups * CUPS_TO_LITERS;
}
```

- Converts cups to liters using a constant conversion factor.

### Tablespoons and Teaspoons to mL
```c
double ConvertVolume(int tablespoons, int teaspoons) {
    const double ML_PER_TEASPOON = 4.92892;
    int totalTeaspoons = (tablespoons * 3) + teaspoons;
    return totalTeaspoons * ML_PER_TEASPOON;
}
```
- Combines both units to teaspoons and converts to milliliters.

### Cost Calculation by Age and Units
```c
int CalculateCost(int age, int units) {
    int cost;
    if (age >= 18 && age < 25) {
        cost = (units > 20) ? (units - 20) * 100 : 0;
    } else {
        cost = (units < 1) ? units * 100 : units * 500 + 200;
    }
    return cost;
}
```
- Demonstrates nested `if` logic and ternary operator use.

### Microwave Bag Size Logic
```c
void PrintPopcornTime(int bagOunces) {

```
- Checks for bag size limits and calculates cooking time.

### Sum of Negative Integers
```c
void OutputResult(int start, int end) {
    int sum = 0;
    for (int i = start; i <= end; ++i) {
        if (i < 0) sum += i;
    }
    printf("%d\n", sum);
}
```

### Find Smallest Until Positive
```c
int FindSmallestVal() {
    int num, smallest;
    scanf("%d", &num);
    smallest = num;
    while (num <= 0) {
        if (num < smallest) smallest = num;
        scanf("%d", &num);
    }
    return smallest;
}
```

### Check All Inputs are Positive Until -99
```c
int InspectVals() {
    int num;
    while (1) {
        scanf("%d", &num);
        if (num == -99) break;
        if (num <= 0) return 0;
    }
    return 1;
}
```
