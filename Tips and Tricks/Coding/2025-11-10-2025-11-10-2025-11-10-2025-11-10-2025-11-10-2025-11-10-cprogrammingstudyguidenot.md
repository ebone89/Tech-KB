### ðŸ“Œ 1. Pass by Pointer vs. Pass by Value

#### Pass by Value
- When you pass a variable to a function by value, a copy is made.
- Modifying the parameter inside the function does **not** affect the original variable.

```c
void ConvHrMin(int timeVal, int hrVal, int minVal) {
   hrVal  = timeVal / 60;  
   minVal = timeVal % 60;
}
```

**Issue:** The `usrHr` and `usrMin` in `main()` remain unchanged.

#### Pass by Pointer (Simulates Pass by Reference)
- You use `&` when calling the function and `*` in the function definition to update the actual argument.

```c
void ConvHrMin(int timeVal, int* hrVal, int* minVal) {
   *hrVal  = timeVal / 60;
   *minVal = timeVal % 60;
}
```

### ðŸ“Œ 3. Working with Arrays in Functions

#### Reading, Modifying, and Printing Arrays

```c
void PrintScoreVals(const int scoreVals[], int numVals) {
   for (int i = 0; i < numVals; ++i) {
      printf("%d\n", scoreVals[i]);
   }
}

void UpdateScoreVals(int scoreVals[], int numVals, int updateVal) {
   for (int i = 0; i < numVals; ++i) {
      scoreVals[i] -= updateVal;
   }
}

int FindScore(const int scoreVals[], int numVals) {
   int returnVal = scoreVals[0];
   for (int i = 1; i < numVals; ++i) {
      if (scoreVals[i] < returnVal) {
         returnVal = scoreVals[i];
      }
   }
   return returnVal;
}

int FindDates(const int datesArray[], int arraySize) {
   int indexFound = -1;
   for (int i = 0; i < arraySize; ++i) {
      if (datesArray[i] == 12 && indexFound == -1) {
         indexFound = i;
      }
   }
   return indexFound;
}
```

### ðŸ“Œ 4. Pointer-Based Value Transformation

```c
void ConvertLength(int totalInches, int* numYards, int* numFeet, int* numInches) {
   *numYards = totalInches / 36;
   totalInches %= 36;
   *numFeet = totalInches / 12;
   totalInches %= 12;
   *numInches = totalInches;
}

void SwapWork(double* val1, double* val2) {
   double temp = *val1;
   *val1 = *val2;
   *val2 = temp;
}

bool AdjustGrade(int score, char* grade) {
   if (score >= 75 && score < 80) {
      *grade = 'B';
      return true;
   } else if (score >= 80 && score <= 120) {
      *grade = 'A';
      return true;
   }
   return false;
}

void IncreaseCarnations(int carnationsArray[], int arraySize, int carnationsAdded) {
   for (int i = 0; i < arraySize; ++i) {
      carnationsArray[i] += carnationsAdded;
   }
}

int SumMangos(const int mangosArray[], int numItems) {
   int sum = 0;
   for (int i = 0; i < numItems; ++i) {
      sum += mangosArray[i];
   }
   return sum;
}

void SwapArrayEnds(int sortArray[], int arraySize) {
   int temp = sortArray[0];
   sortArray[0] = sortArray[arraySize - 1];
   sortArray[arraySize - 1] = temp;
}
```
