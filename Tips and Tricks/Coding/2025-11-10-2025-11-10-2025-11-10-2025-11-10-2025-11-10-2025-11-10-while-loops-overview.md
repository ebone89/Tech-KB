A `while` loop repeatedly executes a block of code **as long as a specified condition is true**.

```c
while (condition) {
    // Code block
}
```

---

## ðŸ” Example 1: Read characters until `'q'` is entered

```c
char inputVal;
scanf(" %c", &inputVal);

while (inputVal != 'q') {
    printf("Input is %c\n", inputVal);
    scanf(" %c", &inputVal);
}

printf("Exit\n");
```

**Concepts:**
- `'q'` is the **sentinel** value.
- `scanf(" %c", &inputVal);` uses a space before `%c` to **skip whitespace** characters.

---

## ðŸ”¢ Example 2: Count characters until `'q'` (excluding `'q'`)

```c
char userIn;
int sumChars = 0;

scanf(" %c", &userIn);

while (userIn != 'q') {
    sumChars = sumChars + 1;
    scanf(" %c", &userIn);
}

printf("%d\n", sumChars);
```

**Concepts:**
- Count increases for each character **that isn't `'q'`**.
- Loop **terminates** when `'q'` is entered.
- `'q'` is **not included** in the total.

## âž•âž– Example 3: Add or subtract integers based on odd/even

```c
int numInput;
int finalVal = 0;

scanf("%d", &numInput);

while (numInput >= 0) {
    if (numInput % 2 != 0) {
        finalVal = finalVal - numInput; // Subtract odd
    } else {
        finalVal = finalVal + numInput; // Add even
    }
    scanf("%d", &numInput);
}

printf("Final value is %d\n", finalVal);
```

**Concepts:**
- Loop continues while `numInput` is **non-negative**.
- Use `% 2 != 0` to check for **odd** numbers.
- **Odd numbers are subtracted**, **even numbers are added**.

---

## âœ… Tips for Using `while` Loops
- Ensure the loop condition **eventually becomes false** to avoid infinite loops.
- Always **update the loop control variable** inside the loop.
- When reading characters, use `scanf(" %c", &ch);` to **skip whitespace** (e.g., newlines).
