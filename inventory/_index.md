# Inventory Index

This folder contains structured descriptions of your homelab infrastructure.

## Files

- [devices.yaml](devices.yaml)  Device inventory for your lab (hosts, VMs, network gear).
- [vlan.yaml](vlan.yaml)        VLAN definitions and associated tags.
- [subnets.yaml](subnets.yaml)  IP subnet definitions and gateway info.
- [services.yaml](services.yaml) Services running in the lab and where they live.

Update these files as you add or remove devices, services, or networks.
